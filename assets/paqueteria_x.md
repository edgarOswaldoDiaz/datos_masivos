### **Paquetería en R para Minería de Datos en X (antes Twitter)**

| **Paquete en R**           | **Descripción detallada**                                                                                                                                                                            | **Funcionalidad específica para X / Twitter**                                                                                                                              | **Aplicación en Sentiment Analysis**                                                                                                         | **Consideraciones técnicas y académicas**                                                                                                                             |
| -------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **rtweet**                 | Paquete especializado para interactuar con la API de X/Twitter desde R. Permite recolectar, procesar y analizar grandes volúmenes de tuits de manera estructurada, respetando los límites de la API. | Autenticación OAuth, descarga de tuits por palabra clave, hashtag, usuario, ubicación geográfica y periodo de tiempo. Obtención de métricas como retuits, likes y replies. | Fuente principal de datos textuales para análisis de sentimiento. Permite crear datasets listos para tokenización y clasificación emocional. | Requiere credenciales de desarrollador en X. Ideal para prácticas de minería de datos reales. Permite discutir temas de ética, privacidad y uso responsable de datos. |
| **tidyverse**              | Conjunto de paquetes (dplyr, tidyr, stringr, ggplot2, etc.) orientados a la manipulación, limpieza y visualización de datos bajo el paradigma “tidy data”.                                           | Limpieza de texto, normalización de caracteres, eliminación de URLs, menciones y emojis provenientes de tuits.                                                             | Preparación del texto previo al análisis de sentimiento (data wrangling). Visualización de resultados emocionales.                           | Fundamental para reforzar buenas prácticas de preparación de datos. Muy alineado con flujos reproducibles en ciencia de datos.                                        |
| **tidytext**               | Paquete que permite transformar texto en formatos “ordenados” (tokens) compatibles con tidyverse, facilitando el análisis estadístico del lenguaje natural.                                          | Tokenización de tuits, eliminación de stopwords, manejo de n-gramas.                                                                                                       | Asociación de tokens con diccionarios de sentimiento (bing, afinn, nrc). Cálculo de polaridad y emociones.                                   | Ideal para introducir NLP clásico en R. Favorece análisis explicables, útiles en contextos académicos y legales.                                                      |
| **textdata**               | Paquete auxiliar para descargar y gestionar léxicos de sentimiento y otros recursos lingüísticos.                                                                                                    | Acceso a diccionarios de sentimiento utilizados sobre texto proveniente de X.                                                                                              | Base para clasificar emociones como alegría, enojo, tristeza, miedo, etc.                                                                    | Permite discutir sesgos lingüísticos y limitaciones de diccionarios genéricos en redes sociales.                                                                      |
| **syuzhet**                | Paquete orientado específicamente al análisis de sentimiento y emociones en textos narrativos y sociales.                                                                                            | Análisis directo del sentimiento de tuits sin necesidad de tokenización manual avanzada.                                                                                   | Obtención de puntajes emocionales y polaridad global de textos cortos.                                                                       | Útil para ejercicios introductorios. Menor control metodológico que tidytext, pero más inmediato.                                                                     |
| **tm**                     | Paquete clásico para minería de texto que ofrece estructuras como corpus y matrices documento-término.                                                                                               | Construcción de corpus a partir de tuits descargados.                                                                                                                      | Análisis exploratorio del contenido textual previo al sentimiento.                                                                           | Más tradicional; permite contrastar enfoques clásicos vs tidy text.                                                                                                   |
| **quanteda**               | Paquete avanzado para análisis cuantitativo de texto, altamente eficiente para grandes volúmenes de datos.                                                                                           | Procesamiento masivo de tuits, creación de matrices de características.                                                                                                    | Integración con modelos estadísticos y de machine learning para sentimiento.                                                                 | Adecuado para proyectos de mayor escala. Requiere mayor curva de aprendizaje.                                                                                         |
| **wordcloud / wordcloud2** | Paquetes para visualización de frecuencia de palabras.                                                                                                                                               | Representación gráfica del vocabulario más usado en tuits.                                                                                                                 | Identificación visual de términos asociados a sentimientos positivos o negativos.                                                            | Útil como complemento visual, no como técnica analítica principal.                                                                                                    |
| **ggplot2**                | Paquete de visualización de datos basado en la gramática de gráficos.                                                                                                                                | Gráficas de evolución temporal del sentimiento en X.                                                                                                                       | Visualización de tendencias emocionales por tiempo, tema o usuario.                                                                          | Refuerza habilidades de comunicación de resultados analíticos.                                                                                                        |
